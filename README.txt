Each directory houses an individual analysis and is organized into 
multiple subdirectories:
* raw_data: This subdirectory type contains the original output files

the manuscript's content.

* code: This subdirectory type contains all the necessary code to 
replicate the analyses from scratch. Depending on the analysis type, it 
may include:

    * awk scripts and scripts from command-line programs
    * SLURM job submission scripts

and directories within these scripts to align with your current system 
configuration.
